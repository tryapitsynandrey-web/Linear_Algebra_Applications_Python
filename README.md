# Linear Algebra and Numerical Methods Applications

This repository contains a Jupyter Notebook demonstrating practical applications of linear algebra and numerical methods using Python and NumPy. The project is divided into four main tasks:

## Project Overview

### 1. Image Processing with Matrices

- **Contrast Adjustment:** Scaling pixel intensities.
- **Brightness Correction:** Adding scalar offsets to matrices.
- **Blending:** Linear combination of two image matrices.

### 2. Recommendation System (Cosine Similarity)

- Implementing a simple content-based recommendation engine.
- Using vector norms and dot products to calculate cosine similarity between user preferences and item features.

### 3. Production Optimization (Systems of Linear Equations)

- Solving a system of linear equations to determine the number of units (drones) to assemble based on component constraints.
- Verification using matrix-vector multiplication.

### 4. Trend Forecasting (Least Squares Method)

- Modeling data trends using the method of least squares.
- Predicting future values (e.g., charge percentage) based on historical time-series data.
- Comparison between `numpy.linalg.lstsq` and the manual normal equation approach.

## Requirements

- Python 3.x
- NumPy

## How to Use

1. Open the notebook in Google Colab or a local Jupyter environment.
2. Run the cells sequentially to see the analytical solutions and their programmatic implementations.
